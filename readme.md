# Projeto Pizzaria / Full Stack – Em Desenvolvimento

## Sobre o projeto
Projeto pessoal desenvolvido com foco no aprendizado prático de programação e desenvolvimento web. O projeto busca consolidar conceitos de front-end e iniciar estudos em back-end, simulando um ambiente real de desenvolvimento.

## Objetivo
Aprofundar conhecimentos em lógica de programação, estruturação de código e integração entre front-end e back-end, aplicando boas práticas de desenvolvimento.

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript (em aprendizado)
- Node.js (em aprendizado)
- React (em aprendizado)
- SQL Server (em aprendizado)

## O que estou aprendendo com este projeto
- Estruturação semântica com HTML
- Estilização e responsividade com CSS
- Fundamentos de JavaScript
- Organização de código e lógica de programação
- Conceitos iniciais de back-end com Node.js
- Noções de banco de dados e consultas SQL

## Status do projeto
Em desenvolvimento – novas funcionalidades serão adicionadas conforme avanço nos estudos.
